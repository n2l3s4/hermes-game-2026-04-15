# Bobo's Monster Band 🎵

A musical memory game for Bobo! Watch the colorful monsters sing their notes, then repeat the pattern by clicking them.

## How to Play
1. Click anywhere to start
2. Listen carefully as the monsters light up and sing
3. Click the monsters in the same order
4. Complete all 5 rounds to win!
